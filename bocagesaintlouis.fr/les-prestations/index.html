RESTAURATION

Une restauration soignée dans un cadre agréable

Un personnel compétent et attentif vous propose un service de qualité. La cuisine réalisée sur place est servie à l’assiette joliment dressée sur table individuelle.

Chaque semaine, la Direction le chef de cuisine, établissent les menus du jour.

Une carte de remplacement vous est également proposée, ainsi qu’un menu gourmet.

Le soir une petite carte a été mise en place afin de mieux vous satisfaire. Le repas peut vous être apporté dans appartement si vous le désirez. Dans ce cadre, vous recevrez votre famille ou vos invités. L’équipe est à votre écoute pour imaginer des menus exceptionnels.

Si votre santé l’exige, un régime particulier vous sera proposé après une rencontre avec la direction.

Une bonne table dans un cadre chaleureux et convivial et une prestation de qualité sont les soucis de toute l’équipe de la résidence. Menu type
